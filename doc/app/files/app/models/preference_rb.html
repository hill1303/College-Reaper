The Preference class represents the preferences expressed by a user. These include importance of distance, class start time, class end time, waitlisting, maximum number of classes and explicit courses.

A single attribute will store this information: * :choices is a hash of all the user's preferences, stored as an hstore column so as to be query-able as well
